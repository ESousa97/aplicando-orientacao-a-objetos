# Security Policy

> [!WARNING]
> Este repositório está arquivado e não está mais ativo.
> Ele permanece público somente para fins de estudo.
> Não há garantia de resposta, revisão ou correção, inclusive para relatos de segurança.

## Supported Versions

| Versão | Suporte |
|---|---|
| main | supported |

## Reportando vulnerabilidades

Para reportar vulnerabilidades, utilize GitHub Security Advisories do repositório:

- https://github.com/enoquesousa/aplicando-orientacao-a-objetos/security/advisories

Não reporte falhas de segurança em issues públicas.

## SLA de resposta

- Confirmação inicial: até 72 horas.
- Análise técnica: até 7 dias corridos.
- Correção ou mitigação inicial: até 30 dias, conforme severidade.
