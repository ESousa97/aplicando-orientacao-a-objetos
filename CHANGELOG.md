# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning.

## [1.0.1] - 2026-02-21

### Changed
- Repositório marcado como arquivado para estudo, sem manutenção ativa.
- Dependabot configurado para não abrir novos PRs.
- Documentos de governança e templates de issue/PR atualizados com aviso visível de arquivamento.

## [1.0.0] - 2026-02-21

### Added
- Scripts de automação local para build, lint, test e validate.
- Testes unitários em Java puro para `Filme` e `Musica`.
- Estrutura de governança no GitHub: CI, templates de issue/PR, CODEOWNERS e Dependabot.
- Arquivos de política: CONTRIBUTING, SECURITY e CODE_OF_CONDUCT.
- Documentação técnica de arquitetura e setup.

### Changed
- Correção de naming de classe pública em `Aula01/Desafio/Atividade3/MetodoMusica.java`.
- Padronização de configuração de repositório (`.editorconfig`, `.gitignore`, `.gitattributes`, `.env.example`).

### Security
- Inclusão de varredura de secrets em workflow agendado.
